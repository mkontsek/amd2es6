Transform AMD to ES6 for code files using AST Transformations
tested only in Chrome

This app helps with transforming AMD files to CommonJS format. The output does not account for ESlint rules, copy/paste it into your IDE and correct the errors before commiting to Git

  1. Copy and paste the entire AMD file contents to the text area named 'Source'
  2. Click on 'Transform'
  3. Preview transformation in Output area
  4. Copy and past output from text area next to the Output area, you cannot directly copy/paste from output area.

For correct migration into a class structure, you need to have the return statement inside of the define()method in the following format: 

              /**
                * @class ActiveInstallmentCollection
                */
              return Backbone.Collection.extend({ ...
